# Руководство по монетизации Flymate через Travelpayouts

## Обзор

Flymate интегрирован с партнёрской программой **Travelpayouts** (официальная партнёрка Aviasales). Вы получаете комиссию от каждой покупки билета, совершённой по вашим партнёрским ссылкам.

## Как это работает?

1. **Вы регистрируетесь** в Travelpayouts и получаете уникальный **Marker ID**
2. **Бот добавляет ваш маркер** во все ссылки на покупку билетов
3. **Пользователь переходит** по ссылке из уведомления бота
4. **Пользователь покупает билет** на Aviasales
5. **Вы получаете комиссию** от продажи (обычно 1-3% от стоимости билета)

## Шаг 1: Регистрация в Travelpayouts

1. Перейдите на https://www.travelpayouts.com/
2. Зарегистрируйтесь как партнёр
3. Подтвердите email и заполните профиль
4. Пройдите в раздел **"Инструменты"** → **"API"**
5. Найдите ваш **Marker ID** (обычно выглядит как `12345.your_username`)

## Шаг 2: Добавьте маркер в .env файл

Откройте файл `bot/.env` и добавьте/обновите переменную:

```env
TRAVELPAYOUTS_MARKER=12345.your_username
```

**ВАЖНО:** Замените `12345.your_username` на ваш реальный маркер из личного кабинета Travelpayouts.

## Шаг 3: Перезапустите worker

После добавления маркера перезапустите фоновый процесс проверки цен:

```bash
# Остановите текущий worker (Ctrl+C)
# Запустите заново
cd bot
python worker.py
```

## Как проверить, что всё работает?

1. Создайте тестовую подписку через бота
2. Дождитесь уведомления о найденном билете
3. Проверьте ссылку в уведомлении:
   - **Ссылка "🔗 Купить билет"** должна содержать `?marker=ВАШ_МАРКЕР` или `&marker=ВАШ_МАРКЕР`
   - **Ссылка "🔎 Найти похожие"** должна содержать `?marker=ВАШ_МАРКЕР`

Пример правильной ссылки:
```
https://aviasales.ru/search/MOW1205LED21?marker=12345.username
```

## Как отслеживать статистику и выплаты?

1. Войдите в личный кабинет Travelpayouts
2. Раздел **"Статистика"** → посмотрите количество кликов и конверсий
3. Раздел **"Финансы"** → ваш баланс и история выплат

### Минимальная сумма для вывода:
- **$50** для большинства способов вывода
- Выплаты обычно раз в месяц

## Примерная доходность

Реальные цифры зависят от многих факторов, но вот примерная модель:

### Базовые показатели:
- **Комиссия:** 1.5-3% от стоимости билета (зависит от направления и авиакомпании)
- **Конверсия:** 2-5% (из 100 переходов покупают 2-5 человек)
- **Средний чек:** $200-400 (зависит от направлений)

### Пример расчёта:

**Сценарий 1: Небольшая аудитория (100 активных пользователей)**
- Уведомлений в день: 200-300
- Переходов по ссылкам: 50-100 (25-30% CTR)
- Покупок: 1-3 в день (конверсия 2-3%)
- Средний чек: $250
- Комиссия: 2% = $5 за билет
- **Доход в месяц: $150-450**

**Сценарий 2: Средняя аудитория (500 активных пользователей)**
- Уведомлений в день: 1000-1500
- Переходов по ссылкам: 250-450
- Покупок: 5-15 в день
- **Доход в месяц: $750-2250**

**Сценарий 3: Большая аудитория (2000+ активных пользователей)**
- Уведомлений в день: 4000-6000
- Переходов по ссылкам: 1000-1800
- Покупок: 20-60 в день
- **Доход в месяц: $3000-9000**

## Факторы, влияющие на доход

### Позитивные факторы:
✅ Качество подборки (точные уведомления = выше конверсия)
✅ Популярные направления (Москва-Сочи, Москва-СПб)
✅ Сезонность (перед праздниками и отпусками)
✅ Реальная экономия (чем больше скидка, тем выше конверсия)
✅ Быстрые уведомления (пользователь успевает купить до исчезновения цены)

### Негативные факторы:
❌ Слишком много спама (пользователь отписывается)
❌ Ложные срабатывания (цена уже изменилась)
❌ Экзотические направления (низкий спрос)
❌ Высокие цены в уведомлениях (пользователь не видит выгоды)

## Техническая информация

### Где добавляется маркер:

1. **bot/worker.py:365** — добавление маркера в deeplink от API
   ```python
   link = build_deeplink(min_offer.get("link"), marker=settings.TRAVELPAYOUTS_MARKER)
   ```

2. **bot/worker.py:366** — добавление маркера в search URL
   ```python
   search_link = build_search_url(origin, destination, dep, marker=settings.TRAVELPAYOUTS_MARKER)
   ```

3. **bot/settings.py:15** — хранение маркера в настройках
   ```python
   TRAVELPAYOUTS_MARKER: str = Field("", description="Travelpayouts affiliate marker ID for monetization")
   ```

### Формат ссылок:

**Deeplink от API:**
```
https://aviasales.ru/flights/MOW-LED?...параметры...&marker=ВАШ_МАРКЕР  (для русского языка)
https://aviasales.com/flights/MOW-LED?...параметры...&marker=ВАШ_МАРКЕР  (для английского языка)
```

**Search URL:**
```
https://www.aviasales.ru/search/MOW0512LED21?marker=ВАШ_МАРКЕР  (для русского языка)
https://www.aviasales.com/search/MOW0512LED21?marker=ВАШ_МАРКЕР  (для английского языка)
```

**Автоматический выбор домена:**
Бот автоматически определяет домен (.ru или .com) на основе языка пользователя, указанного в поле `language` таблицы `users`:
- `language = "ru"` → `aviasales.ru`
- `language = "en"` → `aviasales.com`
- По умолчанию используется `ru`

## Частые вопросы

**Q: Почему маркер не отображается в ссылках?**
A: Проверьте, что переменная `TRAVELPAYOUTS_MARKER` установлена в `.env` и worker перезапущен.

**Q: Будут ли пользователи платить больше из-за моего маркера?**
A: Нет! Цена для пользователя не меняется. Комиссию платит Aviasales из своей прибыли.

**Q: Сколько действует cookie после клика?**
A: Обычно 30 дней. Если пользователь кликнул по вашей ссылке, но купил билет через неделю, вы всё равно получите комиссию.

**Q: Можно ли использовать несколько маркеров?**
A: Технически да, но это против правил Travelpayouts. Используйте только один маркер на один проект.

**Q: Когда придут первые деньги?**
A: Обычно через 30-60 дней после первой продажи (время подтверждения бронирования + время обработки выплаты).
