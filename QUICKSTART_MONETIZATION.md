# Быстрый старт монетизации Flymate

## 3 простых шага до первых денег

### Шаг 1: Регистрация в Travelpayouts (5 минут)

1. Перейдите на **https://www.travelpayouts.com/**
2. Нажмите **"Регистрация"** (или "Sign Up")
3. Заполните форму:
   - Email
   - Пароль
   - Страна
   - Веб-сайт/Telegram канал (можно указать: `@your_bot_username`)
4. Подтвердите email
5. Заполните профиль партнёра (тип трафика: "Telegram Bot")

### Шаг 2: Получите Marker ID (1 минута)

1. Войдите в личный кабинет Travelpayouts
2. Перейдите: **Инструменты** → **API** (или **Tools** → **API**)
3. Найдите поле **"Marker"** (ваш уникальный партнёрский идентификатор)
4. Скопируйте его (формат: `12345.username`)

### Шаг 3: Добавьте маркер в бота (30 секунд)

1. Откройте файл `bot/.env`
2. Найдите строку:
   ```env
   TRAVELPAYOUTS_MARKER=
   ```
3. Вставьте ваш маркер:
   ```env
   TRAVELPAYOUTS_MARKER=12345.username
   ```
4. Сохраните файл
5. Перезапустите worker:
   ```bash
   # Остановите текущий процесс (Ctrl+C)
   cd bot
   python worker.py
   ```

## ✅ Готово! Теперь вы зарабатываете

Все ссылки на покупку билетов автоматически содержат ваш партнёрский маркер.

## Как проверить?

1. Создайте тестовую подписку в боте
2. Дождитесь уведомления о билете
3. Проверьте ссылку — она должна выглядеть так:
   ```
   https://aviasales.ru/search/MOW0512LED21?marker=12345.username
   ```

## Отслеживание статистики

- **Клики:** Travelpayouts → Статистика → Клики
- **Продажи:** Travelpayouts → Статистика → Конверсии
- **Доход:** Travelpayouts → Финансы → Баланс
